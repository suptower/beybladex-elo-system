What is the ELO Density Map?
This visualization shows how ELO ratings are distributed across all Beys and how that distribution changes over time.
  • Histogram: See how many Beys fall into each ELO range at a specific point in time.
  • KDE Evolution: Watch the distribution curve evolve from early matches (blue) to recent (red).
  • Density Heatmap: A 2D view showing density across both ELO and time dimensions.
  • Statistics: Track meta-level metrics like spread, skewness, and mean/median trends.
