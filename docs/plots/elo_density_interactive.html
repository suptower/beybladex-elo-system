What is the ELO Density Map?
This visualization shows how ELO ratings are distributed across all Beys and how that distribution changes over time.
  • Histogram: See how many Beys fall into each ELO range.
  • KDE Evolution: Watch the distribution curve evolve over time.
  • Density Heatmap: A 2D view showing density across ELO and time.
  • Statistics: Track meta-level metrics like spread and skewness.
